INTRODUÇÃO

React é uma biblioteca Javascript para criação de interfaces para usuário.

A princípal função do React é organizar o conteúdo que será mostrado ao usuário.

Mesmo podendo ser usado de outras formas(veremos mais adiante), o princípal uso do React em dia é para criar sites, ou interfaces web.

O React é uma das princípais ferramentas do mercado para criação de SPA's. SPA's(Single Page Application) são aplicações web(sites) que tem como objetivo trazer uma experiência similar a de um programa de computador para o usuário(sem os reloads e etc comuns aos sites).

HISTÓRIA DO REACT

Essa biblioteca surgiu em 2011, no facebook, e passou a ser utilizada na interface do mural de notícias da rede social.

No ano seguinte, passou a integrar também a área de tecnologia do instagram e de várias outras ferramentas da empresa.

O react tem sido usado por grandes companhias ao redor do mundo.

Algumas delas: Netflix, Airbnb, American Express, Facebook, Whatsapp, Ebay, Instagram, Globo, Hotmart, Ifood, dentre outras..
